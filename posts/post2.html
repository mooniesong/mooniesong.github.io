GitHub Pages로 블로그 만들기

GitHub Pages를 이용하면 무료로 개인 홈페이지를 만들 수 있습니다. 설정도 어렵지 않아요. 단계별로 따라해 봅시다.

1단계 — 저장소 만들기

GitHub에 로그인한 뒤 새 저장소를 만드세요. 이름은 반드시 username.github.io 형식이어야 합니다. username 자리에 본인의 GitHub 아이디를 넣으면 됩니다.

2단계 — 파일 올리기

저장소 루트에 index.html 파일을 올리면 자동으로 홈페이지가 됩니다. CSS, JS 파일도 함께 올리면 됩니다.

3단계 — GitHub Pages 활성화

  1. 저장소 → Settings 클릭
  2. 왼쪽 메뉴에서 Pages 선택
  3. Source를 main 브랜치로 설정
  4. Save 클릭

완료!

몇 분 후 https://username.github.io에 접속하면 내 홈페이지가 나타납니다. 커스텀 도메인도 연결할 수 있어요. 저처럼 직접 HTML로 만들어도 되고, Jekyll 같은 정적 사이트 생성기를 써도 됩니다.
